A small plugin for create clusters of posts.

== Description ==
You can create your clusters an easy way. 

1. Go to Clusters
2. Select the options
3. Copy the shortcode 
4. Paste in your post/page

== Installation ==
Install Pixel Cluster like you would install any other WordPress plugin.

Dashboard Method:

1. Login to your WordPress admin and go to Plugins -> Add New
2. Type \"Pixel Cluster\" in the search bar and select this plugin
3. Click \"Install\", and then \"Activate Plugin\"


Upload Method:

1. Unzip the plugin and upload the \"pixel-cluster\" folder to your \'wp-content/plugins\' directory
2. Activate the plugin through the Plugins menu in WordPress
